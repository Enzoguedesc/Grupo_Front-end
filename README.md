# 5W2H
- 

- What?  
  O que o sistema faz?  

- Why?  
  Por que o sistema existe  

- Who?  
  Quem usaria o sistema?  

- Where?  
  Aonde ele seria usado?  

- When?  
  Quando seria usado ou disponivel para uso?  

- How?  
  Como ele seria feito?  

- How much?  
  Custo de tempo de produçao do sistema.
